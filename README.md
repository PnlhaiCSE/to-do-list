## To-Do List Web App

Một ứng dụng to-do list đơn giản để quản lý công việc hàng ngày.

## Tính năng

- ✅ Thêm công việc mới
- ❌ Xóa công việc theo tên
- 🗑️ Xóa tất cả công việc (Clear All)
-  ✓ Đánh dấu công việc đã hoàn thành
- 🔄 Làm mới trạng thái
- 💾 Lưu dữ liệu trên localStorage (không cần server)
